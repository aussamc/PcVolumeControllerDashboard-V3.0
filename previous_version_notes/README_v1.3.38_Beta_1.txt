PC Volume Controller Dashboard v1.3.38 Beta 1

This beta adds dashboard-side OLED preview and display settings for the upcoming six-display hardware.

Main additions:
- OLED Preview tab with six channel preview panels.
- Display mode setting.
- OLED brightness setting.
- OLED dim/sleep timeout setting.
- Setup import/factory-reset STATE send coalescing.
- Hardware Test SLEEP/WAKE command throttle.

ESP32 firmware note:
- This beta keeps the ESP32 firmware source folder as Computer_Volume_Controller_v1.3.37.1.
- Required ESP32 protocol remains v1.3.37.1.
- Rename/update firmware only when real ESP32 behaviour changes are added.

Included:
- Dashboard source.
- ESP32 firmware source.
- tools/esptool.exe.
- firmware_bin/build_instructions.txt.
