PC Volume Controller - esptool setup
====================================

The dashboard firmware flasher looks for:

tools/esptool.exe

This release could not bundle esptool.exe from the ChatGPT build environment because internet access/toolchain installation is unavailable here.

Recommended Windows setup options:

Option 1 - Python/pip esptool
1. Install Python 3 for Windows.
2. Open PowerShell.
3. Run:
   python -m pip install esptool
4. Find the installed esptool.exe, usually under a Python Scripts folder.
5. Copy esptool.exe into this release's tools/ folder.

Option 2 - Standalone esptool.exe
1. Download a trusted Windows esptool.exe package.
2. Place esptool.exe into this release's tools/ folder.

The dashboard will not flash firmware until tools/esptool.exe exists.
