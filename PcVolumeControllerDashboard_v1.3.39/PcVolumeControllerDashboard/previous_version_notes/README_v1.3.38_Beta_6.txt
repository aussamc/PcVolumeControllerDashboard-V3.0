PC Volume Controller Dashboard v1.3.38 Beta 6

What changed in this beta
- Added click-and-drag slider behaviour across the dashboard sliders.
- Added connected OLED idle behaviour with an adjustable timeout.
- Added connected idle action selection:
  - Turn off display
  - Dim brightness to 30%
- Default connected idle timeout is 10 minutes.
- Added anti-burn-in pixel shifting with the default state enabled.
- Connected OLED idle is based on no controller input and no display state/volume changes from the PC.

Version summary
- Dashboard: v1.3.38 Beta 6
- Required ESP32 protocol: v1.3.38 Beta 6
- Included ESP32 firmware source: Computer_Volume_Controller_v1.3.38_Beta_6

Important
- This beta changes ESP32 firmware behavior.
- Flash the included v1.3.38 Beta 6 firmware before testing connected OLED idle or anti-burn-in behavior.
