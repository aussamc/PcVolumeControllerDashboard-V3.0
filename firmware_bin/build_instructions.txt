ESP32 firmware binary build instructions
======================================

The ESP32 Arduino source project is located at:

  Computer_Volume_Controller_v2.0/
    Computer_Volume_Controller_v2.0.ino

Target board: ESP32-S3-DevKitC-1-N16R8

Required Arduino libraries:
  - Adafruit GFX Library
  - Adafruit SSD1306

To build with Arduino CLI:

1. Install Arduino CLI.
2. Install the ESP32 Arduino core:
     arduino-cli core install esp32:esp32
3. Install required libraries:
     arduino-cli lib install "Adafruit GFX Library"
     arduino-cli lib install "Adafruit SSD1306"
4. Compile:
     arduino-cli compile --fqbn esp32:esp32:esp32s3 --output-dir firmware_bin Computer_Volume_Controller_v2.0

Expected output files:
  - bootloader.bin
  - partitions.bin
  - boot_app0.bin
  - Computer_Volume_Controller_v2.0.ino.bin

Alternatively, open Computer_Volume_Controller_v2.0.ino in Arduino IDE,
select "ESP32S3 Dev Module" as the board, and use Sketch > Export Compiled Binary.
